# Product Vision

A timer that disappears into the background of life.

Users should feel they are interacting with time itself,
not operating software.

# Design Principles

- Inspired by Google Clock on Pixel.
- Material 3 design language.
- Large typography.
- Extreme whitespace.
- Minimal color usage.
- Motion explains state transitions.
- No skeuomorphism.
- No gamification.
- No unnecessary settings.

# Motion

- All animations 150-300ms.
- Use easing curves similar to Material Motion.
- Avoid bounce animation.
- Avoid flashy transitions.

# Typography

English:
- Google Sans / Inter

Chinese:
- Noto Sans SC

# Performance

- 60fps on low-end devices.
- No unnecessary rerenders.
- GPU accelerated animations only.

# Emotion

The app should feel:

calm
precise
trustworthy
quiet
